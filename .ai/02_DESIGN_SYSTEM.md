# 4.1 Typography Philosophy

Typography is one of the strongest elements of GauRakshak Bharat.

Instead of decorative fonts, the platform should use clean, modern, highly readable typography that supports education and long reading sessions.

Every heading should immediately attract attention while every paragraph should remain comfortable to read.

Typography should create hierarchy without creating clutter.

# 5.1 Philosophy

Spacing creates clarity.

A beautiful UI is not created by colors alone.

White space guides the user's attention,
improves readability,
reduces cognitive load,
and creates a premium appearance.

Every margin and padding should follow one unified spacing system.

Never use random spacing values.

---

# Base Grid System

The entire application follows an

8px Grid System.

All spacing values should be multiples of 8 whenever possible.

---
